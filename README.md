# TTKPiano

TTKPiano is a Java Web Application built with JSP/Servlet following the MVC architecture.  
The system allows users to browse music courses, manage shopping carts, and perform user management features.

---

# Features

## Authentication
- Login / Logout
- Session management
- User role checking (Admin/User)

## User Management
- Create user account
- Search users by last name
- Update user information
- Delete users
- View user details

## Shopping Cart
- View all courses/books
- Add items to cart
- Update quantity
- Remove items
- Save cart using Cookie + Session

## Product Management
- View available courses/books
- Display product information
- Category support
